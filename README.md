## Development

Clone this repository:

```sh
git clone https://github.com/mrpatiwi/react-slope-one.git
cd react-slope-one
```

Install dependencies:

```sh
npm install
```

Start on [`http://localhost:3000`](http://localhost:3000) with:

```sh
npm start
```
